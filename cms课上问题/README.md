# cms-
该仓库用来整理河北师范大学软件学院2015级《cms与模板制作》课程内容。本门课程涵盖了项目开发过程中的版本控制工具，开发管理平台，原型工具，也涉及到了UI设计和数据库设计以及使用phpcms进行后台的搭建和开发。涉及内容广、信息量大，为河北师范大学15级学生的后续实训工作做铺垫。
